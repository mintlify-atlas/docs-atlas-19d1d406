## Prerequisites

<Info>
Darwin requires Node.js 20 or higher. Check your version with `node --version`.
</Info>

Before you begin, you'll need:

- Node.js 20+ installed
- An API key for one of the supported models:
  - Google Gemini API key (recommended)
  - Anthropic API key
  - OpenAI API key

## Installation

<Steps>
  <Step title="Navigate to the darwin-sdk package">
    ```bash
    cd packages/darwin-sdk
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>

  <Step title="Set up environment variables">
    Create a `.env` file in `packages/darwin-sdk`:

    ```bash .env
    # For Google models (recommended)
    GOOGLE_GENERATIVE_AI_API_KEY=your_google_api_key_here

    # OR for Anthropic models
    ANTHROPIC_API_KEY=your_anthropic_api_key_here

    # OR for OpenAI models
    OPENAI_API_KEY=your_openai_api_key_here
    ```

    <Tip>
    Get a free Google Gemini API key at [ai.google.dev](https://ai.google.dev)
    </Tip>
  </Step>
</Steps>

## Your First Automation

<Steps>
  <Step title="Initialize configuration">
    ```bash
    npm run dev init
    ```

    This creates a `darwin.config.json` file with default settings.
  </Step>

  <Step title="Edit the configuration">
    Open `darwin.config.json` and customize it:

    ```json darwin.config.json
    {
      "website": "https://example.com",
      "task": "Click the sign up button and fill out the registration form",
      "model": "google/gemini-3-flash-preview",
      "maxSteps": 20,
      "verbose": 2,
      "env": "LOCAL"
    }
    ```

    <Note>
    The `task` field should describe what you want the agent to accomplish in natural language.
    </Note>
  </Step>

  <Step title="Run the agent">
    ```bash
    npm run dev run
    ```

    A browser window will open and you'll see:
    - The agent navigating your website
    - A visible cursor showing where it's clicking
    - Terminal output showing its reasoning
    - Visual overlays for analytics events
  </Step>
</Steps>

## What's Happening?

When you run the agent, Darwin:

1. **Opens a headful browser** - You can watch it work
2. **Navigates to your website** - Loads the page you specified
3. **Thinks before acting** - Uses AI to reason about the page
4. **Takes actions** - Clicks, types, scrolls naturally
5. **Tracks everything** - Captures analytics and thoughts

### Terminal Output

You'll see output like this:

```bash
Initializing Stagehand browser agent...
Stagehand initialized successfully
Navigating to: https://example.com
Page loaded, starting task execution...

💭 Thinking: I see a sign up button in the top right corner

🔧 Action: click

💭 Thinking: The form appeared, I need to fill in the email field

🔧 Action: type

✓ Task completed
  Thoughts captured: 8
```

### Browser Visual Feedback

The browser shows:

- **Timer overlay** - Shows elapsed time
- **Analytics notifications** - When events are tracked
- **Reasoning subtitle** - Current thought process
- **Cursor highlight** - Visual cursor movement

## Override Configuration

You can override config values via command line:

```bash
# Override website and task
npm run dev run -- --website "https://different-site.com" --task "Find the pricing page"

# Override model
npm run dev run -- --model "anthropic/claude-sonnet-4-5-20250929"

# Override max steps
npm run dev run -- --steps 30
```

## Common Issues

<Warning>
If you see "API key not found" errors, make sure your `.env` file is in the correct location (`packages/darwin-sdk/.env`) and contains the right variable name for your chosen model.
</Warning>

### Browser doesn't open

- Make sure you're using `env: "LOCAL"` in your config
- Check that Chrome/Chromium is installed on your system

### Agent gets stuck

- Try increasing `maxSteps` in your config
- Simplify your task description
- Check terminal output for reasoning - it may be confused about the page structure
