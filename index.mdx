Darwin is an AI-powered browser automation framework built on Stagehand that not only automates web interactions but also observes user behavior, analyzes friction points, and automatically evolves your UI/UX to create better experiences.

## Quick Example

Get started with a simple automation task:

<CodeGroup>

```bash CLI
# Initialize configuration
darwin init

# Run an automation task
darwin run --website "https://example.com" --task "Find the contact page"
```

```typescript Programmatic
import { BrowserAgent } from 'darwin-ui';

const agent = new BrowserAgent({
  website: 'https://example.com',
  task: 'Find the contact page',
  model: 'google/gemini-3-flash-preview',
  maxSteps: 20
});

await agent.init();
const result = await agent.execute();
await agent.close();
```

```bash HTTP API
# Start the Darwin API server
cd packages/darwin-sdk/core
npm run start:dev

# Run automation via API
curl -X POST http://localhost:3002/api/run \
  -H "Content-Type: application/json" \
  -d '{
    "website": "https://example.com",
    "task": "Find the contact page"
  }'
```

</CodeGroup>

## Evolution Pipeline

Darwin's unique capability is its evolution pipeline that transforms user behavior into UI improvements:

<Steps>
  <Step title="Observe">
    Darwin automates tasks while capturing detailed analytics about user interactions, pain points, and behavior patterns
  </Step>
  <Step title="Analyze">
    The Analyst engine processes interaction data and identifies UX issues, friction points, and optimization opportunities
  </Step>
  <Step title="Evolve">
    AI-powered code generation automatically implements UI/UX improvements to address identified issues
  </Step>
  <Step title="Iterate">
    Continuous monitoring and improvement cycles create progressively better user experiences
  </Step>
</Steps>
